# Ejecutar en desarrollo

1. Clonar el repositorio
2. Instalar dependencias
   ```bash
    npm install
   ```
3. Tener Nest CLI instalado
   ```bash
     npm i -g @nestjs/cli
   ```
4. Levantar la base de datos con docker-compose
   ```bash
     docker-compose up -d
   ```
5. Levantar el servidor
   ```bash
     npm run start:dev
   ```

## Stack usado:

- Mongo
- NestJS
